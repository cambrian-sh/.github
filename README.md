**Cambrian** _— The OS for Agentic Organizations_

Cambrian is an **agent operating system** — a complete runtime for AI agents with built-in memory, orchestration, browser automation, observation, verification, custom tools, and skills.

Our architecture is inspired by **cognitive science and biology**, drawing from how natural intelligence systems learn, remember, and act. The result is an agent platform that doesn't just execute workflows — it thinks, adapts, and improves.

The core is open source. On top of it, we are building a **company brain** for organizations, a **cloud platform** for developers, and a **consumer app** for daily personal use.

---

## Key Projects

### [core](https://github.com/cambrian-sh/core)

The Cambrian OS kernel — agent runtime, memory, orchestration, and gRPC services. Written in Go. Source available under Elastic License 2.0.

### [cli](https://github.com/cambrian-sh/cli)

Terminal admin companion and installer — SSH management, CI pipeline integration, TUI dashboard, and remote server administration. TypeScript + Ink v7. Apache 2.0.

### [ui](https://github.com/cambrian-sh/ui)

Desktop UI for day-to-day agent management. Built with Tauri + React + TypeScript. AGPL v3.
